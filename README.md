#### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/soloviovmax/cyborgtests-demo.git
   cd cyborgtests-demo
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Install Chromium for Playwright**
   ```bash
   npx playwright install chromium
   ```
